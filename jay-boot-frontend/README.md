# Jay Boot Frontend

基于 `Vue 3 + TypeScript + Vite` 的前端脚手架，已完成基础路由与应用壳层初始化。

## 技术栈

- Vue 3（Composition API）
- Vue Router 4
- Pinia
- Ant Design Vue
- Vite

## 已初始化路由

- `/dashboard` 控制台总览
- `/auth/register` 注册引导
- `/auth/login` 登录与会话
- `/tenant` Workspace 租户
- `/rbac` 角色权限
- `/billing` 订阅计费
- `/apikey` API Key 管理
- `/ai-gateway` AI Gateway
- `/usage` 用量与运营

## 本地命令

```bash
npm install
npm run build
```

> 当前仅搭建脚手架与路由占位页面，不包含业务功能实现。
