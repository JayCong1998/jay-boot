# 用户端官网风格原型（product_ui）

## 使用方式

1. 浏览器打开 `product_ui/index.html`
2. 通过顶部导航进入各页面
3. 点击原型按钮查看交互反馈

## 页面清单

- `index.html`：站点地图与入口
- `pages/home.html`：官网首页
- `pages/features.html`：功能价值
- `pages/pricing.html`：订阅定价
- `pages/auth-login.html`：登录
- `pages/auth-register.html`：注册
- `pages/onboarding.html`：新手引导
- `pages/workspace.html`：创作工作台
- `pages/result-delivery.html`：结果交付
- `pages/history.html`：历史记录
- `pages/checkout.html`：支付页
- `pages/payment-success.html`：支付成功
- `pages/subscription.html`：订阅中心
- `pages/profile.html`：个人中心
- `pages/referral.html`：邀请裂变
- `pages/help.html`：帮助中心
- `pages/not-found.html`：404

## 设计方向

- 官网风格：顶栏导航 + Hero + 价值分段 + 强转化 CTA
- 无左侧导航，贴近真实 SaaS 用户端访问方式
- 重点服务盈利闭环：拉新 -> 激活 -> 付费 -> 留存 -> 裂变
