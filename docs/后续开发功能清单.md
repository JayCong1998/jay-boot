## 后续开发功能清单

### 核心基础（优先级最高）

| 功能 | 说明 |
|------|------|
| **Redis 缓存支持** | 缓存 Token、权限、系统配置等，提升性能 |
| **统一操作日志** | 记录关键业务操作，支持审计追溯 |
| **API 限流/防刷** | 基于 IP/用户的请求频率限制 |
| **参数校验增强** | 统一参数校验工具，减少重复代码 |

### 重要功能（优先级高）

| 功能 | 说明 |
|------|------|
| **数据字典管理** | 管理下拉选项、枚举值等系统配置项 |
| **系统配置管理** | 动态配置项（如站点名称、开关配置） |
| **代码生成器** | 根据数据库表快速生成 CRUD 代码 |
| **邮件服务** | 注册验证、密码重置、通知发送 |

### 扩展功能（按需选择）

| 功能 | 说明 |
|------|------|
| **文件存储服务** | 本地/MinIO/OSS 统一接口 |
| **定时任务调度** | 定时清理日志、数据同步等 |
| **Excel 导入导出** | 批量数据处理 |
| **前端通用组件库** | 表格、表单、弹窗等业务组件封装 |

### 运维监控（可选）

| 功能 | 说明 |
|------|------|
| **健康检查增强** | 数据库连接、Redis 连接状态 |
| **在线用户管理** | 查看当前在线用户、强制下线 |
| **服务器监控** | CPU、内存、磁盘使用情况 |

---

## 建议

作为一个**轻量级脚手架**，建议优先实现 **核心基础** 的 4 个功能，它们是大多数项目都需要的通用能力。
